📺 PlayList-Counter

A YouTube Playlist Length Calculator — know exactly how long your playlists are in hours, minutes, and seconds. Fast ⚡, clean 🎨, and hassle-free ✅.

🚀 Features

Enter any YouTube playlist URL and get:

🎥 Total number of videos

⏳ Total duration of the playlist

📊 Average video length

⚡ Playback speed options (1.25x, 1.5x, 2x)

Works for large playlists (more than 50 videos) using pagination (nextPageToken).

Dark mode support 🌙.

Clean and responsive UI built with React + TailwindCSS.
